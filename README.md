# 🗺 Markers: 여행 계획 짜기 애플리케이션

여행을 가기 전, 혹은 부모님의 여행 일정을 짜던 중 
간단하게 코스를 만들어서 공유할 수 있는 방법이 없어 만들어 보았습니다.

엑셀로 계획표를 만들면 지도를 첨부할 수 없어 경로를 확인할 수 없고, 
지도에 표시하자니 시간이나 메모를 추가로 넣을 수가 없었습니다.

그래서 지도를 넣어 위치를 알 수 있도록 하고 도착 시간이나 추가적인 메모를 기재할 수 있도록 하였습니다.

<br/>

# 로그인하기

<img width="700" alt="스크린샷 2024-02-13 오후 2 26 56" src="https://github.com/kimyziii/plan-marker/assets/72454630/b6542ad0-8e73-448d-928e-1c328aab2037">

가장 첫 화면은 나만의 경로를 관리할 수 있도록 로그인 화면으로 설정했습니다.

Firebase의 Authentication을 사용하여 이메일/비밀번호 및 구글 로그인을 진행합니다.

<br/>

# 여행경로 리스트

<img width="700" alt="스크린샷 2024-02-13 오후 2 32 45" src="https://github.com/kimyziii/plan-marker/assets/72454630/0fd9acc1-c3be-4a55-9378-e98cde4ae7ff">

로그인에 성공하면 첫번째로 보이는 화면입니다.

다른 사용자가 공개적으로 만든 경로 혹은 나의 경로를 확인합니다. 내가 만든 경로는 X 아이콘을 눌러 바로 삭제할 수 있습니다.

생성일자 혹은 여행경로의 이름 가나다순으로 정렬을 할 수 있습니다.

상세보기 버튼을 눌러 디테일 페이지로 이동합니다.

<br/>


# 경로 생성 페이지

<img width="700" alt="스크린샷 2024-02-13 오후 2 28 50" src="https://github.com/kimyziii/plan-marker/assets/72454630/7ab62940-45c9-4fd7-8e3f-40a3c57190a0">

<img width="700" alt="스크린샷 2024-02-13 오후 2 31 05" src="https://github.com/kimyziii/plan-marker/assets/72454630/78d8fa77-ff64-4c00-b94d-6ac5478ae27f">

여행경로를 새로 생성합니다.

장소 검색창에 장소명 혹은 키워드를 입력한 후 추가 버튼을 눌러 지도에 마커를 추가합니다.
해당 위치가 아니라고 확인되면 삭제 버튼을 눌러 마커를 삭제합니다.

혹은 리스트에 있는 삭제 버튼을 눌러 장소를 삭제합니다.
제목 옆 전체삭제 버튼을 누르면 모든 장소를 삭제합니다.

여행 경로의 이름과 각 장소에 시간 및 메모를 추가합니다.
경로의 순서를 바꿔야 한다면 화살표를 통해 이동할 수 있습니다.

모든 정보를 입력했다면 저장 버튼을 누릅니다. 
저장에 성공할 경우 해당 경로의 상세 페이지로 넘어갑니다.
